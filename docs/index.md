# Generador de Contraseñas

## Descripción
Programa Java para generar contraseñas simples, seguras o con reglas personalizadas, y validar su fuerza.

## Características
- ✅ Generación de contraseñas simples
- ✅ Generación de contraseñas seguras
- ✅ Generación personalizada con reglas
- ✅ Validación de fuerza de contraseñas

## Tecnologías
- Java 11+
- Documentación generada con Javadoc
- Sitio creado con MkDocs Material
